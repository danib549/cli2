# CLI2

A collection of developer tools and utilities.

## Contents

### OpenCode-Py

A Python-based CLI agent for local-first coding assistance. Features include:

- File operations (read, write, edit, glob, grep)
- Shell command execution
- Git integration
- Plan/Build mode workflow
- LLM integration (Anthropic, OpenAI)

### Calculator

An HTML-based calculator with keyboard support.

## Setup

```bash
cd opencode-py
pip install -e .
```
